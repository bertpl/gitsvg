# E003 - Unknown op value

## Cause

The line's `op` field has a value that does not match any of the recognized
operations. The parser cannot dispatch to a model and rejects the line.

## Example

```
{"op": "rebase", "branch": "main"}
```

(`rebase` is not a v0.0.x operation — it lives in the operations backlog)

## How to fix

Use one of the recognized op names: `import`, `grid`, `theme`, `branch`,
`commit`, `merge`, `pull_request`, `remove`, `highlight`. Run
`gitsvg schema --list-ops` for the live list.
