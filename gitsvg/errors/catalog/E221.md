# E221 - Arc corner radius out of range

## Cause

The `theme` op set `arc_corner_radius_in_grid_units` above `0.5`.
Branch-off and merge arcs round one corner, anchored at the lane /
row crossing. With a radius beyond half a grid unit, the rounded
corner would push past the midpoint of the adjacent segment,
producing a self-intersecting arc rather than a clean quarter-turn.

## Example

```
{"op": "theme", "arc_corner_radius_in_grid_units": 0.8}
```

## How to fix

Set `arc_corner_radius_in_grid_units` to a value in the `[0, 0.5]`
range. The package default is `0.4` (80% of the half-grid-unit cap).
