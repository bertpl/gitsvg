# E108 - Value not in allowed set

## Cause

A field with an enumerated set of allowed values received something outside
the set. The most common case is `branch.label_side`, which only accepts
`"left"` or `"right"`.

## Example

```
{"op": "branch", "name": "main", "label_side": "top"}
```

(`label_side` must be `"left"` or `"right"`)

## How to fix

Use one of the allowed values listed in the error message. The JSON Schema
for the op enumerates the acceptable values for every constrained field.
