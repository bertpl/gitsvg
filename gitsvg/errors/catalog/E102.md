# E102 - Wrong field type

## Cause

A field has a value of the wrong JSON type. For example, the schema declares
the field as an integer but the line provides a string, or vice-versa.

## Example

```
{"op": "commit", "branch": "main", "msg": "x", "gap": "two"}
```

(`gap` is an integer; `"two"` is a string)

## How to fix

Use the type the field expects. `gitsvg schema <op>` shows the type of each
field; fix the value or quote it correctly.
