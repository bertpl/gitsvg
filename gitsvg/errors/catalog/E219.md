# E219 - Theme font size must be positive

## Cause

The `theme` op set one of `label_font_size`, `branch_label_font_size`,
or `hash_font_size` to zero. Font sizes are strictly positive: zero
collapses the text to nothing visible and propagates to every pill /
label dimension derived from the font size (pill paddings, the pill
corner, label-line padding).

## Example

```
{"op": "theme", "label_font_size": 0}
```

## How to fix

Set the font size to a positive integer. Package defaults are `11`
for the message and pill fonts and `9` for the hash font.
