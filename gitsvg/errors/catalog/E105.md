# E105 - String too short

## Cause

A string field requires at least N characters and the value is shorter.
Free-form text fields (`msg` on `commit`/`merge`, `path` on `import`) all
require at least one character — empty strings are rejected.

## Example

```
{"op": "commit", "branch": "main", "msg": ""}
```

(`msg` must be non-empty)

## How to fix

Provide a non-empty value, or omit the field if it's optional.
